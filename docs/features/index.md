# Features

The engine behind the [windows](../windows/index.md): what nParse+ parses
out of your log and what it does with it.

**Alerts & timers**

- [Triggers](triggers.md) — the full trigger model: search text, regex,
  tokens, zone gating, outputs
- [Built-in triggers](builtin-triggers.md) — the shipped library of ~65
  raid/combat/utility triggers
- [Chat-command timers](chat-timers.md) — `StartTimer-45-Label` from any
  chat channel
- [Respawn & zone timers](respawn-timers.md) — per-NPC respawn countdowns
  on kill
- [Text-to-speech](tts.md) — spoken alerts on every platform

**Combat**

- [CH chains](ch-chains.md) — complete-heal chain parsing and your-slot
  warnings
- [Combat tracking](combat.md) — the DPS engine, death-loop detection,
  pets, random rolls, faction/exp guessing

**World**

- [Boats](boats.md) — boat schedules from sightings
- [Night Vision fix](night-vision.md) — the community shader/sky fix,
  applied and reverted safely

**Network & accounts**

- [Sharing & PigParse](sharing.md) — map dots, shared timers, and feeds,
  interoperating with EQTool users; plus the self-hostable nparse mode
- [Friends sync](friends-sync.md) — one merged friends list across your
  characters

**App**

- [Self-updater](updater.md) — release checks and per-platform updates
